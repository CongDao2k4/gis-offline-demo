# Hướng dẫn làm Tile Maker biến đổi dữ liệu từ OSM sang Vector.

Mục tiêu cần làm: biến dữ liệu thô thành các lớp bản đồ vector (đường xá, sông)

- Video hướng dẫn Tilemaker: `Vector Tiles Without the Stack - State of the Map` 
    - Nội dung: Video hướng dẫn chính thức từ hội thảo OSM -> xem cách chạy cấu hình file JSON và viết logic lọc thuộc tính bằng ngôn ngữ lập trình Lua. 
- Kênh thực hành cấu hình Map: Kênh YouTube Chaim Krause
    - Nội dung: tìm danh sách video của kênh này, tìm các từ khóa về Tilemaker hoặc OpenMapTiles để xem các bài thực hành chỉnh sửa file cấu hình chi tiết.

## 1. Cấu trúc thư mục cần

- chạy lệnh `mkdir -p tilemaker output`

```text
gis-offline-demo/
├── data/
│   ├── hanoi.osm.pbf
│   └── hanoi_roads.osm.pbf
├── tilemaker/
│   ├── config.json
│   └── process.lua
├── output/
└── scripts/
```

## 2. Tạo file tilemaker/config.json

chạy lệnh `nano tilemaker/config.json`

Viết nội dung này:
```json
{
  "layers": {
    "roads": {
      "minzoom": 6,
      "maxzoom": 14
    }
  },
  "settings": {
    "minzoom": 6,
    "maxzoom": 14,
    "basezoom": 14,
    "include_ids": false,
    "combine_below": 10,
    "name": "hanoi_roads",
    "version": "1.0",
    "description": "Simple Hanoi roads vector tiles generated by Tilemaker",
    "compress": "gzip"
  }
}
```

## 3. Tạo file Lua để chạy `tilemaker/process.lua`

chạy lệnh `nano tilemaker/process.lua`

Nội dung:
```lua
function init_function()
end

function node_function(node)
end

function way_function()
    local highway = Find("highway")

    if highway ~= "" then
        Layer("roads", false)
        Attribute("class", highway)

        local name = Find("name")
        if name ~= "" then
            Attribute("name", name)
        end

        local surface = Find("surface")
        if surface ~= "" then
            Attribute("surface", surface)
        end

        local lanes = Find("lanes")
        if lanes ~= "" then
            Attribute("lanes", lanes)
        end
    end
end
```

Nếu way có tag highway  -> đưa vào layer roads -> giữ class = highway -> giữ thêm name/surface/lanes nếu có

## 4. Chạy Tilemaker bằng Docker + tạo scripts SH chạy

- Muốn chỉ chạy với fie road của hà nội thì dùng:

```bash
docker run --rm -it \
  -v "$PWD":/data \
  ghcr.io/systemed/tilemaker:master \
  /data/data/hanoi_roads.osm.pbf \
  --output /data/output/hanoi_roads.mbtiles \
  --config /data/tilemaker/config.json \
  --process /data/tilemaker/process.lua
```

- Muốn chạy với toàn bộ dữ liệu hà nội, thì dùng:

```bash
docker run --rm -it \
  -v "$PWD":/data \
  ghcr.io/systemed/tilemaker:master \
  /data/data/hanoi.osm.pbf \
  --output /data/output/hanoi.mbtiles \
  --config /data/tilemaker/config.json \
  --process /data/tilemaker/process.lua
```

- Kiểm tra kết quả có chưa: `ls -lh output/` để thấy `hanoi_roads.mbtiles`
- Kiểm tra metadata:
```bash
sqlite3 output/hanoi_roads.mbtiles ".tables"
sqlite3 output/hanoi_roads.mbtiles "SELECT * FROM metadata;" 
```

### Có thể tạo file sh chạy tự động docker với 1 file cụ thể - file `scripts/04_generate_mbtiles.sh`

```bash
#!/bin/bash
set -e

INPUT_FILE="data/hanoi_roads.osm.pbf"
OUTPUT_FILE="output/hanoi_roads.mbtiles"
CONFIG_FILE="tilemaker/config.json"
PROCESS_FILE="tilemaker/process.lua"

echo "[INFO] Checking input files..."

ls -lh "$INPUT_FILE"
ls -lh "$CONFIG_FILE"
ls -lh "$PROCESS_FILE"

mkdir -p output

echo "[INFO] Removing old output if exists..."
rm -f "$OUTPUT_FILE"

echo "[INFO] Generating MBTiles with Tilemaker..."

docker run --rm -it \
  -v "$PWD":/data \
  ghcr.io/systemed/tilemaker:master \
  /data/"$INPUT_FILE" \
  --output /data/"$OUTPUT_FILE" \
  --config /data/"$CONFIG_FILE" \
  --process /data/"$PROCESS_FILE"

echo "[INFO] Generated:"
ls -lh "$OUTPUT_FILE"

echo "[INFO] MBTiles metadata:"
sqlite3 "$OUTPUT_FILE" "SELECT * FROM metadata;"

echo "[DONE] Vector Tile Pipeline completed."
```

- Chạy nó lên: `sh scripts/04_generate_mbtiles.sh`
